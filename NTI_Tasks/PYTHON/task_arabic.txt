شرح الفكرة العامة (الفهم الكامل للتاسك)

البرنامج اللي مطلوب منك يعمل التالي:

ياخد من المستخدم:

IP أو hostname بتاع السيرفر

username

password

اسم directory (المجلد) اللي هيتعامل معاه على السيرفر

يتصل بالسيرفر عبر SSH

يعني هتستخدم مكتبة مثل paramiko

بعد الاتصال، لازم تتأكد إن كل حاجة شغالة وتقدر تنفذ أوامر هناك

يتأكد من وجود فولدر باسم المستخدم دخله

لو مش موجود → ينشئه

لو موجود → يعرض رسالة إنه موجود

يرسل ملف requirements.sh إلى السيرفر

داخل الفولدر اللي أنشأه

بعد كده يشغل السكربت هناك

السكربت requirements.sh هيعمل الآتي على السيرفر:

ينشئ فولدر اسمه logs لو مش موجود

يدخل جوه الفولدر

ينشئ 26 ملف a.log, b.log, ..., z.log

كل ملف يحتوي على الحرف المقابل ليه

يطبع “Logs created successfully.”

بعد ما السكربت يخلص، البرنامج الرئيسي هيكمل:

يسحب ملفات اللوج من السيرفر وينسخها محليًا داخل فولدر جديد اسمه copied_logs

ينشئ ملف copied_files.txt فيه أسماء الملفات اللي نسخها

يطبع رسالة “All done successfully.”
